# Inklight
Book Reviewing Website with Review Sentiment Analysis.

## Project Setup

1. Create a virtual environment
```bash
python -m venv venv
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run django server
```bash
cd inklight
python manage.py runserver
```

4. Migrate database

5. Create superadmin
```
python manage.py createsuperuser
```
6. Login to the admin panel using super user privilage.

7. Import books data in database from `book_small_authors_v3.csv`
